Vào trang:
👉 https://dashboard.ngrok.com/get-started/your-authtoken

Copy Authtoken

Sau đó trong CMD gõ:

ngrok config add-authtoken YOUR_TOKEN

Nếu báo success là ok.

✅ BƯỚC 2: Chạy project Vite của bạn

Vào thư mục project (nơi có package.json):

npm run dev -- --host

Bạn sẽ thấy:

Local:   http://localhost:5173
Network: http://192.168.x.x:5173

👉 Port là 5173
